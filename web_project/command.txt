##Commands used in the project 
django-admin startproject web_project-->  for making project
django-admin startapp hello-->            for making app inside the project
python manage.py runserver-->             for running the server
python manage .py makemigrations-->       for migrating
python manage.py migrate-->               main command for migrating 
python manage.py createsuperuser-->       for creating the superuser
cd ..-->                                 for moving in the behind directory 
cd <folder_name/file_name>--->           for moving in forward folder or file                     
mkdir-->                                  for making the folders
python -m vnev .myvenv-->                 for creating the virtual environment
